# 小林无忧生图

一个本地网页工具，用 Express 代理调用 VModel Seedream 4.5 API。

## 功能

- 上传本地参考图或填写公网参考图片 URL 后执行图生图
- 不上传图片时执行文生图
- 支持 2K / 4K / 自定义尺寸
- 图生图时可手动选择宽高比
- 网页直接填写 API Key，并保存在浏览器中
- 支持多个任务队列并发生成，失败后自动重试
- 自动保存历史记录，可直接用历史提示词和参考图重新生图
- 上传前自动压缩图片，降低上传失败概率
- 队列支持暂停、继续、取消和失败重试
- 支持提示词收藏夹和多行批量提示词任务
- 支持当前队列/历史结果批量下载
- 历史记录支持搜索和按图生图、文生图、日期筛选
- 生成前会展示任务参数预览，确认后再提交
- 参考图可填写用途备注，例如脸、服装、姿势、风格
- 错误信息会按 API Key、余额额度、图片图床、网络和 VModel 分类提示
- 支持多用户/账号切换，不同用户隔离 API Key、历史、队列、收藏和素材库
- 支持每日配额限制、今日提交/成功/失败统计和单图成本估算
- 支持结果图继续作为参考图二次生图
- 支持参考图和结果图对比查看
- 支持提示词增强、负面提示词、任务完成通知
- 支持拖拽排序参考图、本地素材库、历史导入导出
- 支持清理本地 uploads 上传文件和管理员状态面板
- 支持筛选历史批量改参数重跑和下载命名规则
- 支持下载时添加文字水印
- 支持 SQLite 本地数据库同步，换浏览器可恢复同一用户数据
- 支持项目/文件夹、标签、结果评分收藏和素材去重
- 支持上传前裁剪/旋转、参考图角色设置、提示词变量批量展开
- 支持结果 ZIP 打包下载、任务调试面板、提示词版本管理和参数预设
- 支持可选访问密码和 PWA 桌面安装模式
- 自动轮询任务状态并展示结果图
- 支持打开原图、复制链接、直接下载图片
- API Key 可放在本地 `.env`，也可在网页填写并保存在当前浏览器
- 支持一次本地上传最多 5 张参考图；本地图片会先转换成 VModel 可访问的 HTTP 图片 URL，再作为 `image_input` 提交

## 使用

1. 复制环境变量模板：

```powershell
Copy-Item .env.example .env
```

2. 编辑 `.env`，填入你的 VModel API Key：

```env
VMODEL_API_KEY=你的_vmodel_api_key
# 可选：启用网页访问密码
APP_PASSWORD=你的访问密码
```

注意：`.env.example` 只是模板，服务实际读取的是 `.env`。

3. 安装依赖并启动：

```powershell
npm.cmd install
npm.cmd start
```

如果 PowerShell 拦截 `npm.ps1`，也可以直接运行：

```powershell
node server.js
```

也可以直接运行 `start-server.cmd`。

4. 打开：

```text
http://localhost:3000
```

## 接口

- `GET /api/config` 检查本地配置
- `POST /api/create` 创建生成任务
- `GET /api/status/:taskId` 查询任务状态

## 说明

图生图模式下，前端提交 `prompt` 和 `image_input`。Seedream 4.5 的 `image_input` 最多 5 张。VModel API 的文件输入需要公网 HTTP/HTTPS 图片 URL，不接受本地文件路径、localhost 或 `data:image/...;base64,...`。文生图模式下才会提交尺寸和宽高比。
